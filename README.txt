# How To run this Code

## Prerequisites
Before you can run the ReactJS application, make sure you have the following prerequisites installed on your machine:
- Node.js (version 14 or later)
- npm (Node Package Manager, typically comes with Node.js installation)

## Cloning or Downloading the Application
To get started, follow these steps:

1. Open your preferred command-line interface or terminal.
2. Navigate to the directory where you want to clone or download the application.
3. Clone the repository by running the following command:
   ```
   git clone <repository_url>
   ```
   Alternatively, you can download the application's source code as a ZIP archive from the repository.

## Installation
Once you have the application source code on your machine, follow the steps below to install the required packages:

1. Open your command-line interface or terminal.
2. Navigate to the root directory of the downloaded/cloned application.
3. Run the following command to install the required packages:
   ```
   npm install @react-jvectormap/core @react-jvectormap/world
   npm install axios
   npm install react-dom
   ```
   This command will use npm to download and install the necessary packages defined in the application's package.json file.

## Running the Application
After completing the installation steps, you are ready to run the ReactJS application. Follow these steps:

1. Make sure you are still in the root directory of the application in your command-line interface or terminal.
2. Run the following command to start the application:
   ```
   npm start
   ```
   This command will start the React development server and launch the application in your default web browser.
